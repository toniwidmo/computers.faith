ChaosHex

In the late 1990s, Agent M wrote the first version of ChaosHex in MSDOS batch files and QBASIC. That version has been lost, probably forever unless Agent M ever bothers to get data recovery on that pile of old hard drives she has sitting around.

The Linux version

In 2013 she wrote a second version, from scratch, using Linux BASH scripting. The code for this version has been preseved on SourceForge.

You can Download Chaoshex and install on any computer or shell that supports BASH scripting provided you have the correct permissions.
